# Reset Password Task

## Front-End
- This project is built using **React Vite** for the front end.
- **Lucide Icons** package is used for icons.
- The UI is styled using **Tailwind CSS**.
- **React Router DOM** is used for navigating between pages.
- **React Toastify** is used for displaying popup notifications for success and error messages.
- **Axios** is used to handle API requests efficiently.

## Back-End
- **Express.js** framework is used to build the backend.
- **Mongoose** is used to connect and interact with the **MongoDB** database.
- **Nodemailer** is used to send password reset emails to users.
- **Bcrypt** is used to securely hash and store passwords.
- **CORS** package is used to restrict unauthorized access to the backend by allowing only trusted URLs.
- The project follows the **MVC (Model-View-Controller)** folder structure pattern.
